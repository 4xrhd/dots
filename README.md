# dots

Personal dotfiles configuration repository featuring Vim and shell setup.

## Overview

This repository contains my personal configuration files (dotfiles) for a Linux development environment, focusing on Vim customization and shell scripting.

**Language Composition:**
- Vim Script: 55.2%
- Shell: 44.3%
- Vim Snippets: 0.5%

## Contents

### Shell Configuration Files
- **`.bashrc`** - Bash configuration with aliases, functions, and environment setup
- **`.bash_profile`** - Bash profile for login shells
- **`.zshrc`** - Zsh shell configuration with extended features
- **`.xinitrc`** - X Window System initialization
- **`.inputrc`** - GNU Readline configuration for command-line editing

### Vim Configuration
- **`.vimrc`** - Main Vim configuration file
- **`.vimrc2`** - Alternative Vim configuration
- **`.vim/`** - Vim plugin directory
- **`.vim_runtime/`** - Vim runtime configuration
- **`.viminfo`** - Vim state and history

### Application Configuration
- **`.goomwwmrc`** - Goomwwm window manager configuration
- **`.gtkrc-2.0`** - GTK2 theme configuration
- **`.radare2rc`** - Radare2 (reverse engineering) configuration
- **`.gf/`** - Grep filter patterns directory
- **`.oho/`** - Custom configuration directory

### Utilities and Scripts
- **`setup.sh`** - Setup script for initializing dotfiles
- **`optimize-fire.sh`** - Optimization utility script
- **`powerline.sh`** - Powerline shell prompt configuration
- **`ffuf_comletaion.sh`** - Bash completion for ffuf (web fuzzer)
- **`bin/`** - Custom executable scripts directory
- **`tut/`** - Tutorials and documentation directory

### Configuration Lists
- **`mybins.list`** - List of custom binaries and tools
- **`parrot.list`** - Parrot OS specific configuration
- **`code_block.conf`** - Code block syntax configuration
- **`grub`** - GRUB bootloader configuration
